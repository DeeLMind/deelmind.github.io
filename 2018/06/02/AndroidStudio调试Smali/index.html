简介: 利用Android Studio 调试 Smali

安装smalidea 插件

在Android Studio里面安装 smalidea 插件



打开反编译工程

首先找到 JEB 或者 改之理 或者 Android Killer 反编译APK的工程目录



用 Android Studio 打开这个目录



然后在需要的地方设置断点



调试模式打开APK

先在JEB里面找到包名




adb shell am start -D -n com.example.administrator.test/.MainActivity

转发端口

adb shell
ps






adb forward tcp:8888 jdwp:11396

配置Android Studio

设置和我们上一步转发的端口一样的端口8888



然后调试即可
